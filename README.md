# SO Fundraising Platform

GitHub-ready Vite + React + TypeScript app for:
- Squares games
- Event tickets / seats / tables
- Raffles with bundle offers
- Buyer PDF receipts
- Admin purchase data

## Quick start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

- Framework preset: Vite
- Build command: `npm run build`
- Output directory: `dist`

## Notes

This is a client-side preview app scaffold ready to push to GitHub.
For production payments and persistent data, add:
- authentication
- database/storage
- payment provider integration
- server/API routes
